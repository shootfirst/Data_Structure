### 自底向上整合树结果（树形dp）

有很多对树上信息进行统计的问题，可以抽象为树的后序遍历，在后续遍历过程中，将子树的答案交给父亲节点，由父亲节点进行整合再往上传递，根节点的结果就是最终的答案，这就是树形dp，递推方向为自底向上
