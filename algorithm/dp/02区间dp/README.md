# 区间动态规划

状态的定义和转移都和区间有关，称为区间动态规划，dp[i][j] = 区间[i..j]上原问题的解

状态转移：

+ o(1)时间复杂度

一般是与 dp[i + 1][j], dp[i][j - 1], dp[i + 1][j - 1] 有关

+ o(n)时间复杂度

枚举i...j分割点，dp[i][j] = g(f(dp[i][k], dp[k + 1][j])) 其中 k = i .. j-1

区间动态规划一般是用单串问题上，以区间 [i, j] 为单位思考状态的设计和转移
