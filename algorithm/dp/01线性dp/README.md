# 线性动态规划


+ 状态定义：dp[n] := [0..n] 上问题的解

+ 状态转移：dp[n] = f(dp[n-1], ..., dp[0])

大问题规模只与小问题有关，问题规模使用一个变量i表示，i从小到大推导，即得到问题的解

分为四类：单串，双串，矩阵，无串
