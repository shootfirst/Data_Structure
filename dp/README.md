# dp

动态规划最重要的三个概念：

最优子结构，无后效性，重复子问题

动态规划解题步骤：

+ 状态定义(定义子问题)

+ 状态转移(状态转移方程的确定)

+ dp数组初始化

+ 顺序确立

+ 举例推导(用于肉眼debug)

动态规划问题可以细分为以下几类，其中线性dp是最简单的
